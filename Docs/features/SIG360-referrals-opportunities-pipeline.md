# SIG360 Referrals, Opportunities, and Pipeline

## Purpose

This module tracks prospects, opportunities, lead sources, referrals, professional partners, and pipeline stages.

## Opportunity Pipeline Stages

- New Lead
- Contacted
- Discovery Scheduled
- Discovery Completed
- Proposal / Planning Review
- Follow-Up Needed
- Became Client
- Nurture
- Not a Fit
- Lost
- Archived

## Opportunity Types

- Wealth Management
- Retirement Planning
- Insurance Planning
- Investment Review
- Event Lead
- Referral Lead
- Business Owner Planning
- Existing Client Expansion

## Opportunity Fields

- Opportunity Name
- Client/Prospect
- Household
- Opportunity Type
- Stage
- Estimated Value
- Probability
- Assigned Advisor
- Lead Source
- Referral Source
- Next Step
- Next Step Due Date
- Notes
- Status
