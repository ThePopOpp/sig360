# SIG360 Booking, Appointment, and Event Management

## Purpose

This module manages appointment scheduling, meeting types, advisor booking pages, client reminders, internal prep tasks, event pages, RSVPs, check-in, and follow-up workflows.

## Appointment Features

- Advisor booking pages
- Team booking pages
- Meeting type templates
- Internal-only appointment creation
- Client-facing booking links
- Calendar sync
- Meeting link storage
- Appointment reminders
- Appointment status tracking
- Meeting prep generation
- Follow-up workflow trigger

## Reminder Rules

Appointment reminders:

- Confirmation email immediately
- Reminder email 24 hours before
- Optional SMS reminder 24 hours before
- Optional SMS reminder 1 hour before

Event reminders:

- RSVP confirmation
- One week reminder
- One day reminder
- Day-of reminder if appropriate
- Post-event thank-you
