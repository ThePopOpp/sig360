# SIG360 Dashboards and Reporting

## Purpose

This document defines role-based dashboards and reporting for SIG360.

Each user should see the information most relevant to their work.

## Leadership Dashboard

Show:

- Active clients
- New leads
- Upcoming appointments
- Reviews due
- Open opportunities
- Event performance
- Campaign performance
- Team workload
- Overdue tasks
- AI meeting summaries needing review

## Advisor Dashboard

Show:

- Today’s meetings
- Meeting prep briefs
- Assigned clients
- Open tasks
- Reviews due
- Follow-ups due
- Recent client activity
- Draft follow-up emails

## Client Service Dashboard

Show:

- Service tasks
- Document requests
- Appointments needing confirmation
- Forms submitted
- Client follow-ups
- Workflow steps due

## Marketing Dashboard

Show:

- Active campaigns
- Email performance
- SMS performance
- Event RSVPs
- Recent leads
- Segment counts
- Content calendar

## Compliance Dashboard

Show:

- AI content needing approval
- Communication review queue
- SMS consent changes
- Audit log activity
- Document access reports

## Client Portal Dashboard

Show:

- Advisor team
- Upcoming appointment
- Open client tasks
- Requested documents
- Recent meeting summary
- Upcoming events

## Reports

Recommended reports:

- Client List Report
- Household Report
- Advisor Book of Business
- Appointments Report
- Meetings Completed Report
- Tasks by User
- Overdue Tasks
- Workflows in Progress
- Event Attendee Report
- Campaign Engagement Report
- SMS Consent Report
- AI Notes Approval Report
- Audit Log Report
