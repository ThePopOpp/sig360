# AI.SIG360 Docs — Source of Truth

## Purpose

This folder is the working documentation package for the SIG360 webapp. These files are designed to be dropped into VS Code and used by Codex/Claude Code as the planning and implementation source of truth.

SIG360 is the internal and client-facing Strategic Income Group platform for relationship management, appointments, events, financial planning workflows, planner/team operations, marketing communications, documents, tasks, and AI-assisted automation.

## Naming Rules

Use `SIG360-` at the beginning of every major planning document so these files stay grouped together in VS Code.

Use clear feature-based names rather than vague names like `notes.md`, `features.md`, or `dashboard.md`.

## Build Intent

The platform should feel like a modern financial services operating system, not a generic CRM. It should support:

- Strategic Income Group staff operations.
- Financial planner and advisor workflows.
- Client-facing experiences.
- Booking and appointment management.
- Event RSVP and event attendance workflows.
- Client lifecycle management.
- Investment, insurance, wealth management, retirement, and planning workflows.
- Marketing, email, and SMS communications.
- Internal task/project management.
- AI-assisted automation through agents, realtime voice, and structured tools.

## Important Compliance Reminder

SIG360 should not send financial, insurance, investment, or tax advice through automated marketing or AI responses unless the content is approved, compliant, and properly scoped. AI features should help summarize, organize, draft, route, and prepare information, but final advice and recommendations should remain with licensed Strategic Income Group professionals.
